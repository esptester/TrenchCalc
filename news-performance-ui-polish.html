Performance & UI Polish

Published Aug 9, 2025 · 3–5 min read · Category: Release

Our latest update isn’t about new features — it’s about making everything you already use smoother, faster, and easier on the eyes.

1. Faster Input Handling

Input lag in older browsers and lower-powered devices has been reduced by optimising how calculations are triggered. This means fields respond instantly, even with complex trench configurations.

2. Improved Dark Mode Contrast

We’ve refined color variables for dark mode to improve readability in low-light conditions. This makes text, buttons, and diagrams pop without causing glare during night work.

3. Cleaner Card Layouts

Product and news cards now have better spacing, consistent icon sizes, and subtle hover effects to make scanning information quicker.

4. Under-the-Hood Optimisations

Smaller image file sizes, reduced DOM complexity, and more efficient CSS rules keep page loads under a second in most conditions.
